# 机会分析模块说明书

## 模块职责
根据用户的背景信息，匹配 3 个最适合的赚钱方向，帮助用户找到符合自身情况且合规可行的门路。

## 输入要求
- 用户身份：上班族 / 宝妈 / 学生 / 待业 / 创业者 / 其他
- 技能列表
- 兴趣列表
- 可投入时间：每天 < 2 小时 / 2-4 小时 / 4-8 小时 / 全职
- 可投入预算：0 元 / 1000 元内 / 1 万内 / 5 万内 / 5 万以上
- 风险承受能力：保守型 / 均衡型 / 激进型
- 已有资质（可选，如教师证、医师证、心理咨询师证等）

## 第一步：硬性预筛（必须先做）
匹配前先按用户的预算和时间筛掉不可能的方向：

| 用户预算 | 不可推荐的赛道 |
|---|---|
| 0 元 / 1000 元内 | 实体店、加盟、跨境电商、传统养殖、连锁餐饮 |
| 1 万内 | 重资产连锁、自建制造、大型货运 |
| 1 万到 5 万 | 全职大型创业（必须保留至少 6 个月生活费） |

| 用户时间 | 不可推荐的赛道 |
|---|---|
| 每天 < 2 小时 | 直播带货（每天 4 小时起播）、实体店（全职坐店）、跑活类外勤 |
| 每天 2-4 小时 | 重运营 MCN、需要轮班的项目 |

合规红线赛道（医疗 / 法律 / 心理咨询 / 金融 / 教育培训 / 直播带货特定品类）未持证用户一律不推荐，并在输出明确告知「需先取得 ___ 资质再进入」。

## 第二步：匹配规则
### 匹配权重
1. 已有技能：40%
2. 兴趣爱好：30%
3. 职业背景：20%
4. 通用推荐：10%

### 人群适配规则
| 人群 | 优先 | 排除 |
|---|---|---|
| 大学生 | 0 成本 / 碎片时间 / 可积累经验 | 全职 / 高投入 / 需自由职业经验 |
| 家庭主妇/主夫 | 居家 / 时间灵活 / 兼顾家庭 | 固定坐班 / 长时间外出 |
| 刚失业人群 | 1 个月内见现金流 / 低风险 | 长周期 / 高风险 |
| 创业失败人群 | 复用旧资源 / 轻资产重启 | 大额投入 / 高杠杆 |
| 上班族 | 业余可执行 / 不抢工作时间 / 轻资产 | 全职 / 固定营业时间 |

### 推荐方向必须包含的多样性
3 个推荐里至少覆盖 2 种以上类型，避免 3 个都是「内容创作」这类同质方案：
- 内容 / IP 类（公众号 / 短视频 / 小红书等）
- 服务 / 技能变现类（咨询 / 教培 / 设计接单等）
- 实物 / 电商类（闲鱼 / 小批量电商等）
- 数字产品类（模板 / 工具 / 课程等）
- 资源整合类（社群 / 中介 / 分销等）

### 不可作为主推荐的方向
- 快递、外卖、保安、滴滴等纯出卖时间的工作（仅可作为短期过渡补充提示）
- 任何承诺保收益的项目（基本是骗局）
- 拉人头、传销结构的"机会"

## 输出结构
每个推荐方向必须包含：
- 门路名称
- 综合评分（0-10 分）
- 类型标签（内容 / 服务 / 电商 / 数字产品 / 资源整合）
- 匹配理由（必须引用用户提供的至少 1 个具体技能 / 兴趣）
- 核心优势（3 条）
- 投入要求：时间 + 资金 + 是否需要资质 + 是否需要团队
- 预期收益：按赛道周期给出第 1 / 第 3 / 第 6 个月的现金流预期（农业等长周期赛道改用季度 / 年）
- 变现路径（≥ 3 条，每条带收入区间）
- 冷启动 30 天关键动作（按周拆分）
- 风险提示：常见翻车点 + 平台 / 法规红线 + 止损建议

## 强制要求
- 必须先做硬性预筛，预筛不通过的赛道禁止出现在推荐里
- 每个变现路径有具体收入区间，禁止「年入百万」「月入十万」类无依据表述
- 必须显式告知有合规红线的赛道需要的资质
- 推荐结果不能 3 个全是同一类型
- 备考用户必须包含备考与副业的时间分配建议
